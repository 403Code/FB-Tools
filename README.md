## Install script on Termux
```
pkg update && pkg upgrade
pkg install python git
cd $HOME
git clone https://github.com/IzZy619/FB-Tools
cd $HOME/FB-Tools
python -m pip install -r modules.txt
python setup.py
```

## Run tool
```
cd $HOME/FB-Tools
python run.py
```
