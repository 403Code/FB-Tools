## Install script on Termux
```
pkg update && pkg upgrade
pkg install python git
cd $HOME
git clone https://github.com/MieRebus47/FB-Tools
cd $HOME/FB-Tools
python -m pip install -r modules.txt
python setup.py
```

## Run tool
```
cd $HOME/FB-Tools
python run.py
```

## Updating tool
```
cd $HOME/FB-Tools
git pull
python setup.py
```
